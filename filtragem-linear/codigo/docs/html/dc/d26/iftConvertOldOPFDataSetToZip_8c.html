iftConvertOldOPFDataSetToZip.c File Reference

Description

A program that converts an OPF dataset file from the old format (with field function_number) to the new one (ending with .zip file extension). For data sets saved before Feb 27, 2016 use iftConvertOldOPFDataSetFormatWithoutFunctionNumberToZip.c.

Note
See the source code in iftConvertOldOPFDataSetToZip.c
