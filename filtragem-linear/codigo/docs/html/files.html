 iftComputeBrainGrad.cComputes the gradient for a Brain Image as a linear combination of the gradient from two Input Images. The resulting gradient might be:
(1) only the gradient from the first input brain image(function iftBrainGrad())
(2) a linear combination of the gradient from the first input brain image with the gradient of a second input image (eg. an object map), according to the formula:
grad = (alpha * input_grad_img1) + ((1-alpha) * input_grad_img2)

ps: The option (2) only happens if a second input image is passed for the program

 iftSegmentImageBySLIC-LogReg.cSegments an image with Simple Prob. Atlas, and LogReg trained with supervoxels generated by SLIC. The program segments an image using Prob. Atlas and a LogReg Classifier.
The LogReg is trained with supervoxels generated by SLIC using brightness values ([avg, stdev, min, max]) for each supervoxel.

 iftSegmentImageBySLIC-OPF.cSegments an image with SLIC and OPF. The program segments an image by generating supervoxels for that, and classifying using OPF classifier.
Two OPF classifiers are trained: one for 3D coordinates from the supervoxel's center voxel ([x, y, z]), and other for brightness values of the supervoxel ([avg, stdev, min, max])
