Files

file  iftDataSetsByISF.c
 Generates Supervoxels by ISF for the Input Images and builds two datasets with the supervoxels: 1) multi-label dataset with voxel coords from each supervoxel's centroids/roots as feature vectors. 2) binary dataset (object or background) with brightness information from each supervoxel as feature vectors. PS: It is expected that all images are registered. PS2: The true label of each supervoxel is the one with majority voting + 1 to guarantee that the background (true label 0) will be 1.
 
file  iftDataSetsBySLIC.c
 Generates Supervoxels by SLIC for the Input Images and builds two datasets with the supervoxels: 1) multi-label dataset with voxel coords from each supervoxel's centroids/roots as feature vectors. 2) binary dataset (object or background) with brightness information (only voxel's value) from each supervoxel as feature vectors. PS: It is expected that all images are registered. PS2: The true label of each supervoxel is the one with majority voting + 1 to guarantee that the background (true label 0) will be 1.
 
file  iftSegmentImageBySLIC-LogReg.c
 Segments an image with Simple Prob. Atlas, and LogReg trained with supervoxels generated by SLIC. The program segments an image using Prob. Atlas and a LogReg Classifier.
The LogReg is trained with supervoxels generated by SLIC using brightness values ([avg, stdev, min, max]) for each supervoxel.
The training supervoxel dataset is obtained by the program demo/Superpixels/iftDataSetsBySLIC.c.
 
file  iftSegmentImageBySLIC-OPF.c
 Segments an image with SLIC and OPF. The program segments an image by generating supervoxels for that, and classifying using OPF classifier.
Two OPF classifiers are trained: one for 3D coordinates from the supervoxel's center voxel ([x, y, z]), and other for brightness values of the supervoxel ([avg, stdev, min, max]).
 
file  iftSegmentImageBySLIC-SVM.c
 Segments an image with SLIC and SVM. The program segments an image by generating supervoxels for that, and classifying using OPF classifier.
Two OPF classifiers are trained: one for 3D coordinates from the supervoxel's center voxel ([x, y, z]), and other for brightness values of the supervoxel ([avg, stdev, min, max]).
