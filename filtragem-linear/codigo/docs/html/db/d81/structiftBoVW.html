Description

BoVW descriptor.

Author
Felipe Lemes Galvao
Date
June 26, 2017

The BoVW descriptor stores nFeatsOutput representative feature vectors of size nFeatsInput. Those are used to represent a histogram with nFeatsOuput bins, where new pixels fall into the bin of the closest representative feature vector.
