iftISF.h File Reference

Description

Interface to create ISF instances and use them to perform superpixel segmentation.

Author
Felipe Lemes Galvao
Date
May, 2017

An ISF instance is defined by the iftISFOptions struct. Usage then consists of two steps: building the iftISFOptions and then using it to actually over-segmentate images.
