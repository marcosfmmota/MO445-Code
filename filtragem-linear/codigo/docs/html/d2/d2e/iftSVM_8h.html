Optimized function to classify the samples with status predStatus from the dataset Z with a trained One-versus-All LINEAR SVM classifier.

Author
Giovani Chiachia
Warning
This functions only works with Linear/Pre-Computed SVM classifiers.
Note
Prediction matrix - one sample per row, one svm model per column.
The predicted label of each sample is compared with its true label.
Parameters
svmOne-versus-All Linear SVM used to classify the samples.
ZDataset to be classified.
ZtrainDataset used to train the SVM - it uses the samples with status IFT_TRAIN.
sample_statusStatus of the samples to be classified.
out_pred_matReturn by Reference (it != NULL) the matrix with the predicted values resulting of the SVM classification.
Returns
Number of samples incorrectly predicted.

iftSegmentImageBySLIC-SVM.c.
