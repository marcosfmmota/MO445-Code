Assigned value. All fields shares the same memory address. Since that long type is has memory size greater or equal to all integer datatypes (and their unsigned versions), we've just used an only field to represent integer number. The same holds to double and float, and to pointers.

Warning
THERE IS NO SUPPORT TO LONG LONG AND LONG DOUBLE.
