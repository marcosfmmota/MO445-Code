iftClassifyImageByLogReg.c File Reference

Description

Classifies an Image (all voxels) using a Logistic Regression Classifier.

The Image is converted to a Dataset where each sample is a voxel, and its feature vector is only the voxel's brightness value. The Logistic Regression Classifier is trained by the program iftTrainImageClassifierByLogReg.c (demo/Classification/iftTrainImageClassifierByLogReg.c) The program might also compute the object map for each object's label based on the weights of the dataset's samples.
