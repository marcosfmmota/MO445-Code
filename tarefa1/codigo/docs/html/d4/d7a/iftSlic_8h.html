Computes the (Slic) Distance between a voxel u and a cell's center.

Authors
Samuel Martins, John Vargas
Date
May 24, 2016
Note
If the Input Image is colored, 3 features (LAB) are considered.
Otherwise, only the first feature (gray) is considered.
Likewise, if the Input Image is 3D, the z-axis is used in the distance
Parameters
centerCenter (double) coordinates.
uVoxel u (double) coordinates.
cfeats_centerColor Feature Vector of the cell's Center
cfeats_uColor Feature Vector of the Voxel u
stepSide of the initial regular cell = (step) distance between two initial adjacency cell's center
compactnessCompactness: Relative Importance of color similarity and spatial proximity
spatial_dist_factorPre-computed Distance factor used to multiply the spatial distance.
is_3D_imgTells if the input image is 3D or not.
is_color_imgTells if the input image is colored or not.
Returns
The Slic distance between the voxel u and the cell's center
