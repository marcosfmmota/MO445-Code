iftSampler* iftBalancedRandomSubsampling ( const int *  labels,
int  n_samples,
int  n_iters,
int  n_train 
)

Repeats N times a Balanced Subsampling method.

Balances the classes distribution by setting all classes with the same number of samples X that the smaller one has.
After that, it applies a random sampling by selecting X samples for each classes, resulting in a sampled
dataset with C*X samples, where C is the number of classes.
From this sampled dataset, it applies a new sampling in order to select the number of training samples n_train.
If n_train is greater than C*X, all sampled dataset (C*X samples) will be used.
Repeat all steps for each iteration.

Parameters
labelsLabels from the samples.
n_samplesNumber of Samples.
n_itersNumber of iterations.
n_trainNumber of train. samples.
Returns
The resulting sampler.
