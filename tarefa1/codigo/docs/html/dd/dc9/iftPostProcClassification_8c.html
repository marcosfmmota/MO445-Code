iftPostProcClassification.c File Reference

Description

Applies post-processing filters on object mask and on its corresponding object membership map, if it exists, as obtained from binary voxel classification.

Note
It is suitable for the output of codes iftClassifyImageByLogReg.c and iftClassifyImageByOPF.c in ift/demo/Classification, and has been tested for brain segmentation.
