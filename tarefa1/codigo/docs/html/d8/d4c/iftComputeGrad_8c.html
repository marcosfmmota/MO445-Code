iftComputeGrad.c File Reference

Description

Computes the gradient for a Image as a linear combination of the gradient from two Input Images. The resulting gradient might be:
(1) only the gradient from the first input image(function iftImageBasins())
(2) a linear combination of the gradient from the first input image with the gradient of a second input image (eg. an object map), according to the formula:
grad = (alpha * input_grad_img1) + ((1-alpha) * input_grad_img2)

ps: The option (2) only happens if a second input image is passed for the program.
