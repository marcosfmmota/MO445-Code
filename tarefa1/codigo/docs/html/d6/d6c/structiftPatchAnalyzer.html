iftPatchAnalyzer Struct Reference

Description

Struct that defines an Analyzer of a given pair of Image Patches.

It contains the voxel coordinates of a pair of patches that describes the same (symmetric) region in each brain hemisphere. It also contains a specific feature extractor and a classifier trained for this region to indicate, for example, if the region is healthy or not.
