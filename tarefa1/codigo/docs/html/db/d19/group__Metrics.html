int iftBestSampleScoreSum ( const iftMatrix *  score_matrix,
iftMetricType  metric_type,
double *  out_best_score

It Finds the best Sample Score from a Score Matrix, returning the indice of the sample in the matrix.

Each row i=p and column j=p from the Score Matrix corresponds to the sample.
The Best Score is from the Sample whose sum of all scores is MAXIMUM (for SIMILARITY functions) or MINIMUM (for DISTANCE functions).

Parameters
score_matrixScore Matrix that will used to figure out the best Sample Score.
metric_typeType of the Score Function that generated the considered Score Matrix.
out_best_scoreReference of the variable which will store the score of the Most Similar Sample. If NULL, nothing is returned to it.
Returns
The index of the Most Similar Sample.
