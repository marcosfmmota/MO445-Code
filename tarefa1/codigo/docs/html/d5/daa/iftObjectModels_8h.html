◆ iftEffectiveN4BiasFieldCorrection()

iftImage* iftEffectiveN4BiasFieldCorrection ( const iftImage *  img,
const iftImage *  mask,
iftImage **  out_bias 
)

A good scheme to apply the N4 over the input image in a more effective way.

Firtly, N4 is applied with shrink factor 4 and the default arguments. Then, another N4 is applied with shrink factor 6 and the default arguments, in order to make a fine tunning in the correction.

Parameters
imgImage to be corrected.
maskBinary mask that defines the structure of your interest. If NULL, N4 considers the entire image.
out_biasThe resulting Bias Field Image by the N4 correction. If NULL, it is not considered.
Returns
Corrected image by N4.

Apply a set of Pre-Processing operation on to a Brain MRI Image.

The operations correspond to: (1) N4 Bias Field Correction (2) Median Filter for noise reduction (3) Normalization to range [0, 4095] by saturating to 4095 the 2% of brighter voxels

Parameters
mriBrain MR-Image
apply_n4Apply the N4 Bias Field Correction.
apply_smoothingApply Median Filter.
apply_normalizationApply Normalization
Returns
Pre-processed MR Brain Image.

iftImage* iftN4BiasFieldCorrection ( const iftImage *  img,
const iftImage *  mask,
int  shrink_factor,
iftImage **  out_bias 
)

Apply the N4 algorithm for Inhomogeneity Correct in a MRI Image.

It is used the default parameters, suggested by 3D Slicer tool [2], for N4 [1].
[1] Tustison, Nicholas J., et al. "N4ITK: improved N3 bias correction." IEEE transactions on medical imaging 29.6 (2010): 1310-1320.

Parameters
imgImage to be corrected.
maskBinary mask that defines the structure of your interest. If NULL, N4 considers the entire image.
shrink_factorImage Resampling/Shrinking Factor (1, 2, 3, 4, ...) to decrease the computation time. Shrink factors <= 4 are commonly used.
out_biasThe resulting Bias Field Image by the N4 correction. If NULL, it is not considered.
Returns
Corrected image by N4.

Train an Adaptive Probabilistic Atlases (AdaPro).

Each object is indexed at a position i, so that each one of is parameter (erosion radius, ...) is in the same position of the corresponding array. If the label image with the training voxels <train_voxels_img> is NULL, the model won't be adaptive, only relying on the shape constraints of the probabilistic atlases during segmentation

Parameters
atlas_setSet of Registered Atlases (Label Images) on the template <template_img>.
template_imgTemplate Image (Reference Image Space) where the atlases are registered.
train_voxels_imgImage with the markers chosen on template for linear SVM Training. BG voxels has label 0 and objects 1. If NULL, the model won't be adaptive, only relying on the shape constraints of the probabilistic atlases.
labelsArray with the labels of the target objects for training the AdaPro.
e_radius_arr(Optional) Array with the erosion radius to generate the inner seeds for each object. If NULL, nothing is stored.
d_radius_arr(Optional) Array with the dilation radius to generate the outer seeds for each object. If NULL, nothing is stored.
CParameter C for linear SVM training.
Returns
The trained AdaPro.
